# 자산군의 세계

투자할 수 있는 자산의 종류를 알아봅시다.
각 자산군은 고유한 특성이 있어요.

---

## 주식 (Stocks)

회사의 일부를 소유하는 것입니다.
회사가 성장하면 주가가 오르고, 배당도 받을 수 있어요.

### 특징

| 항목 | 내용 |
|------|------|
| 기대 수익 | 높음 (연 7~10% 장기 평균) |
| 변동성 | 높음 (하루에 ±5%도 흔함) |
| 적합한 투자자 | 장기 투자 가능하고 위험 감내 가능한 분 |

### 종류

**국내 주식**
- 개별 종목: 삼성전자, 네이버, 카카오 등
- ETF: KODEX 200, TIGER 코스피 등

**해외 주식**
- 개별 종목: 애플, 테슬라, 마이크로소프트 등
- ETF: S&P 500, 나스닥 100 등

```
수익과 위험의 관계

     수익 ↑
           │            ★ 주식 (고수익/고위험)
     높음  │
           │
     중간  │     ★ 채권
           │
     낮음  │  ★ 현금
           └─────────────────────→ 위험
              낮음    중간    높음
```

> 🐾 **집사 TIP**: 주식은 날씬한 고양이 같아요.
> 빠르게 뛸 수 있지만, 갑자기 방향을 바꾸기도 하죠!

## 채권 (Bonds)

정부나 기업에 돈을 빌려주고 이자를 받는 것입니다.
만기까지 보유하면 원금 + 이자를 돌려받아요.

### 특징

| 항목 | 내용 |
|------|------|
| 기대 수익 | 중간 (연 3~5%) |
| 변동성 | 낮음 (안정적) |
| 적합한 투자자 | 안정 추구, 은퇴 준비 |

### 종류

**국채** (정부가 발행)
- 가장 안전, 수익률 낮음
- 한국 국채, 미국 국채 등

**회사채** (기업이 발행)
- 국채보다 수익률 높음
- 기업 부도 위험 존재

**채권 ETF**
- 여러 채권에 분산 투자
- KODEX 국고채, TIGER 미국채 등

> 🐾 **집사 TIP**: 채권은 느긋한 노묘 같아요.
> 급하지 않고 꾸준히 간식(이자)을 받아먹죠!

## 현금 및 예금

은행에 예치하거나 MMF(머니마켓펀드)에 투자합니다.

### 특징

| 항목 | 내용 |
|------|------|
| 기대 수익 | 낮음 (연 2~4%) |
| 변동성 | 거의 없음 |
| 장점 | 언제든 사용 가능, 원금 보장 |
| 단점 | 인플레이션에 약함 |

### 역할

- **긴급 자금**: 갑자기 돈이 필요할 때 사용
- **투자 대기 자금**: 좋은 기회를 기다리며 보유
- **심리적 안정**: "현금이 있다"는 안심

```
현금의 적정 비중

┌─────────────────────────────────┐
│ 최소 5~10%는 현금으로 보유하세요! │
│                                 │
│ • 급한 상황 대비                 │
│ • 투자 기회 포착                 │
│ • 마음의 평화                    │
└─────────────────────────────────┘
```

## 대체 자산

전통적인 주식/채권 외의 투자 자산입니다.

### 부동산 (리츠, REITs)

건물에 간접 투자하는 방법입니다.

- **장점**: 임대료 수입, 인플레이션 헤지
- **단점**: 경기 민감, 금리 영향
- **투자 방법**: 리츠 ETF (TIGER 부동산인프라 등)

### 금 (Gold)

오래전부터 가치 저장 수단으로 사용된 귀금속입니다.

- **장점**: 인플레이션 헤지, 위기 시 안전자산
- **단점**: 이자/배당 없음, 보관 비용
- **투자 방법**: 금 ETF, 골드바, KRX 금시장

### 원자재

오일, 농산물, 금속 등의 상품입니다.

- **장점**: 인플레이션 헤지, 분산 효과
- **단점**: 변동성 높음, 이해하기 어려움
- **투자 방법**: 원자재 ETF

> 🐾 **집사 TIP**: 대체 자산은 고양이의 숨은 장난감 같아요.
> 평소엔 잊고 있다가 필요할 때 빛을 발하죠!

## 핵심 정리

1. **주식**: 고수익/고위험, 장기 투자의 핵심
2. **채권**: 안정적, 주식과 반대로 움직이는 경향
3. **현금**: 유동성 확보, 최소 5~10% 보유 권장
4. **대체자산**: 추가 분산, 인플레이션 헤지

자산군을 잘 조합하면 위험은 줄이고 수익은 유지할 수 있어요!
