Meowney와 함께 스마트한 투자 집사가 되어보세요!

```
┌─────────────────────────────────────────────────────┐
│                                                     │

│  자산 등록 → 플랜 설정 → 리밸런싱 → 대시보드 확인   │
```

## 1단계: 자산 등록하기

**자산 목록** 페이지에서 보유 자산을 등록합니다.

### 자산 추가 방법

```
┌──────────────────────────────────────────┐
│           자산 추가 화면                  │
├──────────────────────────────────────────┤
│                                          │
│  종목명:  [KODEX 200           ]         │
│  티커:    [069500.KS           ]         │
│  수량:    [50                  ]         │
│  평균단가: [35,000             ] 원       │
│  카테고리: [국내주식      ▼    ]         │
│                                          │
│              [추가하기]                   │
│                                          │
└──────────────────────────────────────────┘
```

### 주요 필드 설명

| 필드 | 설명 |
|------|------|
| 종목명 | 자산의 이름 (자유롭게 입력) |
| 티커 | 주식/ETF 코드 (입력 시 실시간 가격 조회) |
| 수량 | 보유 수량 |
| 평균단가 | 매수 평균 가격 |
| 카테고리 | 국내주식, 해외주식, 채권, 현금 등 |

> 🐾 **TIP**: 티커를 입력하면 현재가를 자동으로 가져와요!
> 수동 입력도 가능합니다.

### 카테고리 활용

자산을 카테고리로 분류하면 리밸런싱이 편해집니다.

```
추천 카테고리 구성:

├── 국내주식
│   ├── KODEX 200
│   └── TIGER 코스피
├── 해외주식
│   ├── VOO (S&P 500)
│   └── QQQ (나스닥 100)
├── 채권
│   ├── KODEX 국고채
│   └── TLT (미국 장기채)
├── 현금
│   └── CMA/MMF
└── 대체자산
    └── 금 ETF
```

## 2단계: 리밸런싱 플랜 만들기

**플랜 설정** 페이지에서 목표 자산 배분을 정합니다.

### 플랜 생성

```
┌──────────────────────────────────────────┐
│           새 플랜 만들기                  │
├──────────────────────────────────────────┤
│                                          │
│  플랜명:  [안정형 포트폴리오     ]        │
│                                          │
│  [ 배분 추가 ]                           │
│                                          │
└──────────────────────────────────────────┘
```

### 배분 방식 선택

Meowney는 두 가지 배분 방식을 지원합니다.

**1. 개별 자산 배분**

특정 종목에 정확한 비율을 지정합니다.

```
┌────────────────────────────────┐
│ 개별 배분 예시                  │
├────────────────────────────────┤
│ KODEX 200      20%  ██████    │
│ VOO            30%  █████████ │
│ 국고채 ETF     25%  ███████   │
│ CMA            10%  ███       │
│ 금 ETF         15%  █████     │
├────────────────────────────────┤
│ 합계          100% ✓          │
└────────────────────────────────┘
```

**2. 그룹 배분**

카테고리별 비율만 설정합니다. 더 유연해요!

```
┌────────────────────────────────┐
│ 그룹 배분 예시                  │
├────────────────────────────────┤
│ 국내주식       25%  ███████   │
│   └─ (해당 카테고리 자산들이   │
│       자동으로 포함됨)         │
│ 해외주식       35%  █████████ │
│ 채권           30%  ████████  │
│ 현금           10%  ███       │
├────────────────────────────────┤
│ 합계          100% ✓          │
└────────────────────────────────┘
```

> 🐾 **TIP**: 처음에는 그룹 배분이 더 쉬워요!
> 자산이 많아지면 개별 배분으로 세밀하게 관리하세요.

### 플랜 예시

**보수적 투자자 플랜**
```
국내주식      20%  █████
해외주식      20%  █████
채권          50%  █████████████
현금          10%  ███
───────────────────────────
합계         100%
```

**균형 잡힌 플랜**
```

해외주식      35%  █████████
채권          30%  ████████
현금          10%  ███
───────────────────────────
합계         100%
```

**공격적 투자자 플랜**
```
국내주식      30%  ████████
해외주식      50%  █████████████
채권          15%  ████
현금           5%  █
───────────────────────────
합계         100%
```

## 3단계: 리밸런싱 확인하기

**리밸런싱** 페이지에서 현재 상태와 목표의 차이를 확인합니다.

### 리밸런싱 화면 구성

```
┌──────────────────────────────────────────────────────┐
│                    리밸런싱                          │
├──────────────────────────────────────────────────────┤
│                                                      │
│  선택된 플랜: [안정형 포트폴리오 ▼]                  │
│                                                      │
│  ┌────────────────────────────────────────────────┐ │
│  │  자산          현재     목표     차이     제안  │ │
│  ├────────────────────────────────────────────────┤ │
│  │  국내주식      30%  →  25%    +5%p   매도 ▼  │ │
│  │  해외주식      32%  →  35%    -3%p   매수 ▲  │ │
│  │  채권          28%  →  30%    -2%p   매수 ▲  │ │
│  │  현금          10%  →  10%     0%p   유지 ─  │ │
│  └────────────────────────────────────────────────┘ │
│                                                      │
│  리밸런싱 필요 금액:                                 │
│  • 국내주식 매도: 약 50만원                          │
│  • 해외주식 매수: 약 30만원                          │
│  • 채권 매수: 약 20만원                              │
│                                                      │
└──────────────────────────────────────────────────────┘
```

Meowney가 **얼마를 사고팔아야 하는지** 자동으로 계산해줍니다!

### 리밸런싱 제안 읽는 법

| 표시 | 의미 |
|------|------|
| ▲ 매수 | 비중이 목표보다 낮음 → 더 사야 함 |
| ▼ 매도 | 비중이 목표보다 높음 → 팔아야 함 |
| ─ 유지 | 목표와 비슷함 → 그대로 유지 |

### 주요 기능

**자산 추이 차트**
- 시간에 따른 총 자산 변화
- 기간 선택: 1개월, 3개월, 6개월, 1년, 전체

**포트폴리오 도넛**
- 카테고리별 현재 비율
- 목표와 비교하기 쉬움

**리밸런싱 알림**
- 비율이 크게 벗어나면 알려줌
- 클릭하면 리밸런싱 페이지로 이동

## Meowney 활용 팁

### 팁 1: 메인 플랜 설정

자주 쓰는 플랜은 **메인 플랜**으로 설정하세요.
리밸런싱 페이지에서 자동으로 선택됩니다.

### 팁 2: 허용 오차 활용

작은 차이까지 리밸런싱하면 피곤해요.
**±5% 이상 벗어났을 때**만 조정하는 것을 추천합니다.

### 팁 3: 정기 체크

```
추천 루틴:

• 매일: 대시보드에서 총 자산 확인 (1분)
• 매월: 자산 목록 업데이트, 수량 점검 (5분)
• 분기: 리밸런싱 필요 여부 확인 (10분)
• 연간: 플랜 재검토, 목표 비율 조정 (30분)
```

### 팁 4: 신규 자금 활용

월급날 추가 투자할 때는 **비중이 낮은 자산**에 집중하세요.
매도 없이 리밸런싱 효과를 얻을 수 있어요!

## 자주 하는 질문

### Q: 자산을 추가했는데 가격이 안 나와요

**A**: 티커가 정확한지 확인하세요.
- 국내 주식/ETF: `종목코드.KS` (예: `005930.KS`)
- 미국 주식/ETF: 그대로 입력 (예: `VOO`, `AAPL`)

티커 없이 수동으로 가격을 입력할 수도 있어요.

### Q: 플랜 비율이 100%가 안 돼요

**A**: 모든 배분의 합이 100%가 되어야 합니다.
빠진 자산군이 없는지 확인하세요.

### Q: 리밸런싱을 꼭 해야 하나요?

**A**: 의무는 아니지만, 추천합니다!
비율이 **±5~10%** 이상 벗어났다면 조정하는 것이 좋아요.
