# 리밸런싱: 균형을 되찾아라

포트폴리오를 처음 설정한 비율로 되돌리는 작업을 **리밸런싱**이라고 합니다.

---

## 리밸런싱이란?

시간이 지나면 자산 가격이 변해서 처음 정한 비율이 틀어집니다.
리밸런싱은 이 비율을 **원래대로 되돌리는** 작업입니다.

### 예시: 6개월 후의 변화

```
┌─────────────┬─────────────┬─────────────┐
│  처음 설정   │  6개월 후   │ 리밸런싱 후  │
│  (목표 비율) │ (비율 틀어짐) │ (원래대로!) │
├─────────────┼─────────────┼─────────────┤
│ 주식 60%    │ 주식 75%    │ 주식 60%    │
│ ████████    │ ██████████  │ ████████    │
├─────────────┼─────────────┼─────────────┤
│ 채권 30%    │ 채권 20%    │ 채권 30%    │
│ ████        │ ███         │ ████        │
├─────────────┼─────────────┼─────────────┤
│ 현금 10%    │ 현금  5%    │ 현금 10%    │
│ █           │ █           │ █           │
└─────────────┴─────────────┴─────────────┘
               주식 일부 매도
               채권/현금 매수
```

주식이 많이 올라서 비중이 커졌다면,
**주식을 팔고 채권/현금을 사서** 원래 비율로 맞춥니다.

---

## 왜 리밸런싱을 해야 할까?

### 1. 위험 관리

주식 비중이 너무 커지면 포트폴리오 전체의 위험도가 올라갑니다.

```
목표: 주식 60%, 채권 40%

비중이 틀어진 상태:
┌────────────────────────────────┐
│ 주식 80% → 위험도 크게 증가!   │
│ 채권 20%                       │
│                                │
│ 주식이 -20% 하락하면?          │
│ 전체 자산: -16% 손실           │
└────────────────────────────────┘

리밸런싱 후:
┌────────────────────────────────┐
│ 주식 60% → 위험도 원래대로     │
│ 채권 40%                       │
│                                │
│ 주식이 -20% 하락하면?          │
│ 전체 자산: -12% 손실 (완화!)   │
└────────────────────────────────┘
```

### 2. 자동 수익 실현

"비싸게 팔고, 싸게 사는" 효과가 자동으로 발생합니다.

```
리밸런싱의 마법:

1. 주식이 많이 올랐다 → 주식 매도 (비쌀 때 팔기)
2. 채권이 상대적으로 쌌다 → 채권 매수 (쌀 때 사기)

결과: 자연스러운 수익 실현 + 저가 매수!
```

### 3. 투자 원칙 유지

감정에 휘둘리지 않고 **기계적으로** 투자할 수 있습니다.

> 🐾 **집사 TIP**: 고양이가 털 빠짐 시즌마다 그루밍하듯,
> 포트폴리오도 주기적인 정리가 필요해요!

## 리밸런싱 방법

### 방법 1: 정기 리밸런싱

일정한 주기마다 리밸런싱을 실행합니다.

| 주기 | 장점 | 단점 |
|------|------|------|
| 월간 | 세밀한 관리 | 거래 비용 증가 |
| 분기 | 적당한 균형 | 추천! |
| 연간 | 간단함 | 비율이 많이 벗어날 수 있음 |

```
정기 리밸런싱 예시 (분기별):

     1월         4월         7월         10월
      ↓           ↓           ↓           ↓
   리밸런싱    리밸런싱    리밸런싱    리밸런싱
```

### 방법 2: 밴드 리밸런싱

목표 비율에서 **일정 범위(밴드)를 벗어나면** 리밸런싱합니다.

```
목표 비율: 주식 60%
허용 범위: ±5%

┌─────────────────────────────────────┐
│                                     │
│      65% ─ ─ ─ ─ ─ ─ ─ 상한선       │
│            ┌─────────┐              │
│            │ 허용 범위 │              │
│      60% ─ │ (OK 구간) │ ← 목표      │
│            │         │              │
│            └─────────┘              │
│      55% ─ ─ ─ ─ ─ ─ ─ 하한선       │
│                                     │
│  55% 미만 또는 65% 초과 시           │
│  → 리밸런싱 실행!                    │
└─────────────────────────────────────┘
```

**장점**: 필요할 때만 리밸런싱
**단점**: 수시로 체크해야 함

### 방법 3: 하이브리드

정기 체크 + 밴드 리밸런싱을 결합합니다.

```
예시:
• 기본: 분기마다 체크
• 추가: 비율이 ±10% 벗어나면 즉시 리밸런싱

이 방법이 가장 실용적!
```

## 리밸런싱 실행하기

### 매도-매수 방식

비중이 높아진 자산을 팔고, 낮아진 자산을 삽니다.

```
현재 상태:
주식 70% (목표 60%) → 10%p 초과
채권 25% (목표 30%) →  5%p 부족
현금  5% (목표 10%) →  5%p 부족

리밸런싱 액션:
┌────────────────────────────────┐
│ 1. 주식 매도: 10%p 분량        │
│ 2. 채권 매수:  5%p 분량        │
│ 3. 현금 확보:  5%p 분량        │
└────────────────────────────────┘
```

### 신규 자금 활용 방식

추가 투자금이 있다면, 비중이 낮은 자산에 집중 투자합니다.

```
월급에서 100만원 추가 투자 시:

현재 상태:
주식 70% (목표 60%) → 초과
채권 25% (목표 30%) → 부족
현금  5% (목표 10%) → 부족

액션:
┌────────────────────────────────┐
│ 100만원 전액을 채권/현금에 투자 │
│ → 매도 없이 비율 조정!          │
└────────────────────────────────┘
```

이 방법은 매도 없이 비율을 맞출 수 있어 세금과 수수료를 아낄 수 있습니다.

## 리밸런싱 주의사항

### 1. 세금 고려

주식 매도 시 양도소득세가 발생할 수 있습니다.

```
국내 주식: 대주주가 아니면 비과세
해외 주식: 연 250만원 초과 수익 시 22% 과세

→ 해외 주식은 연말에 손익 정리도 고려!
```

### 2. 수수료 고려

잦은 매매는 비용 증가로 이어집니다.

```
ETF 매매 수수료: 약 0.015~0.5%
→ 너무 자주 리밸런싱하면 손해!
```

### 3. 급하지 않게

하루이틀 차이는 큰 의미가 없습니다.
차분하게 진행하세요.

> 🐾 **집사 TIP**: 고양이가 천천히 스트레칭하듯,
> 리밸런싱도 서두르지 말고 여유롭게!

## 리밸런싱 체크리스트

리밸런싱할 때 이 순서를 따르세요:

```
□ 1. 현재 자산 비율 확인
□ 2. 목표 비율과 비교
□ 3. 차이가 허용 범위 내인가?
      → 범위 내: 유지
      → 범위 초과: 리밸런싱 실행
□ 4. 매도할 자산, 매수할 자산 결정
□ 5. 세금/수수료 영향 확인
□ 6. 거래 실행
□ 7. 결과 확인
```
