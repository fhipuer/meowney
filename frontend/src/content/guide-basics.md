# 자산배분, 왜 해야 할까?

## 달걀을 한 바구니에 담지 마라

옛 속담에 "달걀을 한 바구니에 담지 마라"라는 말이 있습니다.
바구니를 떨어뜨리면 달걀이 모두 깨지기 때문이죠.

투자도 마찬가지입니다. 모든 돈을 한 곳에 투자하면,
그 투자가 실패했을 때 모든 것을 잃게 됩니다.

```
    한 바구니               여러 바구니
  ┌───────────┐        ┌───┐ ┌───┐ ┌───┐
  │ 🥚🥚🥚🥚🥚 │   vs   │🥚🥚│ │🥚🥚│ │ 🥚│
  └───────────┘        └───┘ └───┘ └───┘
    위험해요!           안전해요!
```

> 🐾 **집사 TIP**: 고양이가 간식을 여러 곳에 숨겨두듯이,
> 우리도 자산을 여러 곳에 나눠두는 것이 좋아요!

---

## 자산배분이란?

**자산배분(Asset Allocation)** 이란 투자 자금을 여러 종류의 자산에 나누어 투자하는 전략입니다.

예를 들어 1,000만원이 있다면:
- 주식에 600만원 (60%)
- 채권에 300만원 (30%)
- 현금에 100만원 (10%)

이렇게 나누어 투자하는 것이 자산배분입니다.

---

## 왜 자산배분이 중요한가?

### 1. 위험 분산

한 자산이 떨어져도 다른 자산이 버텨줍니다.

```
주식만 100% 보유했을 때:
┌──────────────────────────────────┐
│ 주식 -30% 하락 → 자산 -30% 손실  │
└──────────────────────────────────┘

자산배분 했을 때 (주식 60%, 채권 40%):
┌──────────────────────────────────┐
│ 주식 -30% 하락                    │
│ 채권 +5% 상승                     │
│ → 전체 자산: -16% 손실로 완화!    │
└──────────────────────────────────┘
```

### 2. 안정적인 수익

장기적으로 꾸준한 성과를 기대할 수 있습니다.
한 방에 대박을 노리는 것보다 느리지만 확실합니다.

### 3. 심리적 안정

시장이 폭락해도 "내 자산은 분산되어 있으니까"라는 안심이 됩니다.
패닉셀(공포에 의한 매도)을 방지해줍니다.

---

## 자산배분의 마법: 상관관계

서로 **다르게 움직이는** 자산을 함께 가지고 있으면,
전체 포트폴리오의 변동성이 줄어듭니다.

```
┌─────────────┬─────────────┐
│ 주식이 오를 때 │ 주식이 내릴 때 │
├─────────────┼─────────────┤
│ 주식: ↑↑↑   │ 주식: ↓↓↓   │
│ 채권: →     │ 채권: ↑     │
│ 금:   ↓     │ 금:   ↑↑    │
├─────────────┼─────────────┤
│ 전체: ↑     │ 전체: →     │
│ (완만한 상승) │ (크게 안 떨어짐)│
└─────────────┴─────────────┘
```

이것이 바로 **분산투자의 힘**입니다!

---

## 자산배분 vs 종목 선택

많은 초보 투자자들이 "어떤 주식을 살까?"에 집중합니다.
하지만 연구에 따르면 **투자 성과의 90% 이상이 자산배분에 의해 결정**됩니다.

| 투자 결정 | 성과 기여도 |
|----------|-----------|
| 자산배분 (주식/채권/현금 비율) | 약 90% |
| 종목 선택 (어떤 주식을 살지) | 약 5% |
| 매매 타이밍 (언제 사고 팔지) | 약 5% |

> 🐾 **집사 TIP**: 어떤 주식을 살지 고민하기 전에,
> 먼저 "내 돈의 몇 %를 주식에 넣을까?"를 정하세요!

## 나에게 맞는 자산배분 찾기

자산배분 비율은 개인마다 다릅니다.
아래 요소들을 고려해서 정해야 해요.

### 고려 요소

1. **투자 기간**: 얼마나 오래 투자할 수 있나요?
   - 10년 이상 → 주식 비중 높여도 OK
   - 5년 이내 → 안전자산 비중 높이기

2. **위험 감내 능력**: 얼마까지 손실을 견딜 수 있나요?
   - -20% 손실도 괜찮다 → 공격적 배분
   - -10%도 못 견딘다 → 보수적 배분

3. **투자 목적**: 왜 투자하나요?
   - 노후 대비 → 장기 성장형
   - 목돈 마련 → 안정 성장형

### 대표적인 배분 예시

```
┌─────────────────────────────────────────┐
│ 공격형 (젊은 투자자, 장기 투자)           │
│ ██████████████████░░░░░░░ 주식 80%       │
│ ████░░░░░░░░░░░░░░░░░░░░░ 채권 15%       │
│ █░░░░░░░░░░░░░░░░░░░░░░░░ 현금  5%       │
├─────────────────────────────────────────┤
│ 중립형 (균형 잡힌 투자)                   │
│ ████████████░░░░░░░░░░░░░ 주식 60%       │
│ ██████░░░░░░░░░░░░░░░░░░░ 채권 30%       │
```

## 핵심 정리

1. **자산배분**은 여러 자산에 나눠 투자하는 전략
2. **위험을 분산**하고 **안정적인 수익**을 추구
3. 종목 선택보다 **자산배분이 더 중요**
4. **나에게 맞는 비율**을 찾는 것이 첫걸음
